This script is used for extracting interesting game statistics from Starcraft 2 game replay files (.sc2replay). Following staticstics are supported currently,
- General game information including game result, version, map , length etc.
- Player's name & race
- Building Orders
- Overal building statistics

The script requires the latest version of sc2reader package (https://github.com/GraylinKim/sc2reader).

Example usage:
$ sc2anaylzer example.sc2replay
